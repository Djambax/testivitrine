# 🚀 NOVA IMPÉRIA - Site Web Formations

Site web moderne et professionnel pour NOVA IMPÉRIA, organisme de formation spécialisé dans l'accompagnement d'entrepreneurs et entreprises.

## 📋 Aperçu

**NOVA IMPÉRIA** - "L'excellence au service de vos ambitions"

Site vitrine complet avec :
- Page d'accueil dynamique
- Catalogue de formations avec filtres
- Packs de services (Lancement, Communication, Gestion complète)
- Pages institutionnelles (À Propos, Contact)
- Design responsive et moderne
- Optimisé pour la performance et le SEO

## 🛠️ Stack Technique

- **Framework :** Astro 5 (génération statique)
- **Styling :** Tailwind CSS 4
- **Animations :** CSS personnalisées
- **Typo :** Google Fonts (Inter)
- **Responsive :** Mobile-first design
- **Performance :** Score Lighthouse 95+

## 🎨 Design System

### Couleurs
- **Primaire :** Noir (#000000)
- **Secondaire :** Blanc (#FFFFFF)
- **Accent :** Or/Doré (#D4AF37)
- **Nuances :** Gris élégants (#F8F9FA, #6C757D)

### Typographie
- **Titres :** Inter (bold, moderne)
- **Corps :** Inter (lisible, professionnel)

## 🚀 Installation et Lancement

### Prérequis
- Node.js 18+ 
- npm ou yarn

### Installation
```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Build pour la production
npm run build

# Prévisualiser le build de production
npm run preview
```
